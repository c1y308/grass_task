# 柔顺低附着不平整地形下人形机器人本体感知鲁棒行走的风险门控课程强化学习方法

## 0 中心命题

> **草地类地形下的人形机器人失稳不是瞬时摔倒事件，而是“接触失配—姿态扰动—危险补偿”逐级累积的风险链条；仅以成功率驱动课程晋级会放任高风险策略进入更高难度地形，因此需要风险门控课程学习。**

## 1 问题描述

本文研究 Unitree G1 人形机器人在未知草地类柔顺不平整地形上的本体感知鲁棒行走问题。机器人不使用额外视觉、触觉或外部地形传感器，也不显式估计地形类别。策略输入仅由机器人自身可反馈的本体感知量构成，包括关节位置、关节速度、机身姿态、角速度、历史动作以及可由平台获得的接触相关状态。策略输出为关节期望位置或低层控制器可执行的动作命令。

草地类地形被建模为柔顺、不平整和低摩擦因素耦合的随机环境。该类地形的主要困难不在于单一几何障碍，而在于脚地接触状态与期望步态时序之间存在不确定偏差。例如，摆动腿可能提前接触隐藏凸起，支撑脚可能在低摩擦草面上产生滑移，足端可能因地面柔顺性发生局部下陷。这些扰动进一步诱发机身 roll/pitch 波动，并促使策略使用踝关节或髋关节进行补偿。若补偿动作持续时间长、频率高或接近硬件安全边界，则可能形成不安全的 aggressive compensation。

## 2 草地类耦合地形随机化模型

### 2.1 地形参数向量

定义草地类地形参数向量为

```math
\xi=\{h,\mu,k,c,p_{\mathrm{trans}}\}
```

式中：

$h$ 为地形高度扰动幅值，用于描述草地下隐藏凸起、小坑和小包（触地时序失配）；

$\mu$ 为足地摩擦系数，用于描述干草、湿草和泥草之间的接触差异（支撑相滑移）；

$k $为接触刚度，用于描述地面柔顺性；$c$ 为接触阻尼，用于描述沉陷过程中的能量耗散（足端沉陷与姿态慢性偏移）；

$p_{\mathrm{trans}}$ 为地形切换概率，用于描述 $flat-to-grass、grass-to-flat$ 等地形过渡事件。

### 2.2 地形难度变量

定义课程难度标量

```math
\lambda\in[0,1]
```

随着 $\lambda$ 增大，地形参数范围逐步扩展为

```math
h_{\max}(\lambda)\uparrow,\quad
\mu_{\min}(\lambda)\downarrow,\quad
k_{\min}(\lambda)\downarrow,\quad
p_{\mathrm{trans}}(\lambda)\uparrow
```

其中，$h_{\max}$ 增大表示不平整程度增强，$\mu_{\min}$降低表示更易打滑，$k_{\min}$ 降低表示地面更加柔顺，$p_{\mathrm{trans}}$ 增大表示地形切换更频繁。本文课程学习由该外生地形难度变量驱动，而非由策略当前风险值直接驱动。

## 3 风险门控课程强化学习方法

### 3.1 强化学习框架

基于 unitree_rl_lab 构建并行仿真训练环境，采用 PPO 算法训练人形机器人行走策略。策略输入为本体感知观测历史，输出为关节动作。奖励函数包括速度跟踪、姿态稳定、能耗约束、动作平滑、足端接触合理性等常规项。为避免方法复杂化，本文不将全部风险诊断指标直接并入奖励函数，而是将其主要用于课程晋级和评价分析。

### 3.2 风险门控晋级准则

第 `i` 个课程阶段的地形参数范围由 $\lambda_i$ 决定。当策略在当前阶段满足成功率和风险门控条件时，课程进入下一阶段：

```math
S_i \geq S^*,\quad
R_{\mathrm{contact}} \leq R_{\mathrm{contact}}^*,\quad
R_{\mathrm{posture}} \leq R_{\mathrm{posture}}^*,\quad
R_{\mathrm{comp}} \leq R_{\mathrm{comp}}^*
```

式中，$S_i$ 为当前阶段成功率，$S^*$为成功率阈值；$R_{\mathrm{contact}}、R_{\mathrm{posture}}、R_{\mathrm{comp}}$ 分别为接触风险、姿态风险和补偿风险统计量。该设计使策略不能仅依靠“未摔倒”通过阶段，而需要在足底滑移、姿态波动和危险补偿方面同时满足基本安全条件。

## 4 风险分解指标

### 4.1 接触风险

接触风险用于描述期望接触时序与真实接触事件之间的不一致，可定义为

```math
E_{\mathrm{contact}} =
\left|t_{\mathrm{touchdown}}^{\mathrm{real}}-
t_{\mathrm{touchdown}}^{\mathrm{expected}}\right|
```

并进一步统计 touchdown error、foot slip ratio、stance duration deviation、unexpected contact、missed/delayed support 和 contact-window IoU。该类指标用于判断策略是否真正学会适应草地类地形接触不确定性。

真实触地时刻$t_{\mathrm{touchdown}}^{\mathrm{real}}$可以从仿真 contact force 或 foot contact flag 获得。

但 $t_{\mathrm{touchdown}}^{\mathrm{expected}}$有如下方法：

第一，用平地 reference gait 的相位来定义。
 第二，用策略内部 gait phase 或 command phase 来定义。
 第三，用上一段稳定步态统计得到期望支撑相窗口。
 第四，用足端高度下降到阈值附近的时间作为预测触地窗口。

如果不讲清楚，touchdown error 会显得像人为构造的指标。目前拟采用期望接触窗口的构造：

定义每条腿在相位 $\phi$ 下的期望支撑窗口，例如：

​												$W_l^{exp}=[t_{l,on}^{exp},t_{l,off}^{exp}] $

真实接触事件为：

​												$W_l^{real}=\{t∣F_{l,t}^z>F_{min}\}$

然后接触风险不要只用一个 touchdown error，而应由四项组成：

​											$$ R_{contact} = w_1 E_{td} + w_2 E_{slip} + w_3 E_{unexpected} + w_4 E_{missed} $$

其中：

​												$$ E_{slip} = \frac{1}{T_s}\sum_{t\in stances} \mathbf{1}\left(\left\|v_{foot,t}^{xy}\right\|>v_{slip}\right) $$ 

​												$$ E_{unexpected} = \frac{1}{T}\sum_{t} \mathbf{1}\left(F_{foot,t}^z>F_{min},\, t\notin \mathcal{W}^{exp}\right) $$

​												$$ E_{missed} = \frac{1}{|\mathcal{W}^{exp}|}\sum_{t\in \mathcal{W}^{exp}} \mathbf{1}\left(F_{foot,t}^z<F_{min}\right) $$

其中，$\mathcal{W}^{exp}$ 表示所有腿期望支撑窗口的集合。$E_{unexpected}$ 描述期望支撑窗口之外的提前接触或异常接触，$E_{missed}$ 描述期望支撑窗口内真实接触不足、延迟支撑或漏支撑现象。前者对应“过早接触隐藏凸起”，后者对应“该支撑时没有形成有效支撑”，二者共同刻画草地类地形下接触时序与接触充分性的失配。

为避免 $E_{td}$ 只反映单一触地时刻误差，还可以在评价分析中报告期望接触窗口与真实接触窗口的一致性：

```math
\mathrm{IoU}_{contact}=
\frac{|W_l^{exp}\cap W_l^{real}|}
{|W_l^{exp}\cup W_l^{real}|}
```

$\mathrm{IoU}_{contact}$ 越高表示真实接触越集中于期望支撑窗口内。本文可将其作为接触一致性的诊断指标，用于补充解释 $R_{contact}$ 的变化，而不一定直接纳入课程晋级门控。

### 4.2 姿态风险

姿态风险用于描述草地扰动引起的机身 roll/pitch 波动，可定义为

```math
E_{\mathrm{posture}} =
\mathrm{RMS}(\theta_{\mathrm{roll}})
+\mathrm{RMS}(\theta_{\mathrm{pitch}})
+\max \left|\dot{\theta}_{\mathrm{base}}\right|
```

同时统计 base roll/pitch RMS、base angular velocity、CoM height fluctuation 和 terrain transition 后的 recovery time。

### 4.3 补偿风险

补偿风险用于区分必要的柔顺地形补偿和不安全的剧烈补偿。基础补偿能量可写为

```math
E_{\mathrm{comp}} =
\sum_t \|\tau_t\|^2+
\sum_t \|\Delta q_t^{\mathrm{ankle}}\|^2
```

但本文不以动作幅值是否降低作为唯一安全标准，而进一步评价补偿质量。相位对齐指标定义为

```math
C_{\mathrm{phase}} =
\frac{
\text{compensation energy within valid stance-response window}
}{
\text{total compensation energy}
}
```

补偿效率定义为事件触发的诊断指标，而不是课程晋级门控的主条件。具体地，以 slip、unexpected contact 或 missed support 事件发生时刻 $t_0$ 为中心，定义补偿前基线窗口和补偿后恢复窗口：

```math
\mathcal{T}_{pre}=[t_0-\Delta t_{pre},t_0),\quad
\mathcal{T}_{post}=[t_0,t_0+\Delta t_{post}]
```

在固定时间窗内计算补偿前后的风险变化：

```math
\Delta R_{\mathrm{contact}} =
\bar{R}_{\mathrm{contact}}(\mathcal{T}_{pre})
-
\bar{R}_{\mathrm{contact}}(\mathcal{T}_{post})
```

```math
\Delta R_{\mathrm{posture}} =
\bar{R}_{\mathrm{posture}}(\mathcal{T}_{pre})
-
\bar{R}_{\mathrm{posture}}(\mathcal{T}_{post})
```

则补偿效率可写为

```math
\eta_{\mathrm{comp}} =
\frac{
\Delta R_{\mathrm{contact}}+\Delta R_{\mathrm{posture}}
}{
E_{\mathrm{joint}}
}
```

其中，$E_{\mathrm{joint}}$ 为 $\mathcal{T}_{post}$ 内的关节力矩、关节动作或动作增量能量。若 $\eta_{\mathrm{comp}}$ 较高，说明单位关节补偿代价能够带来更明显的后续接触风险和姿态风险下降；若 $\eta_{\mathrm{comp}}$ 较低甚至为负，则说明补偿动作虽然发生，但没有有效削弱后续风险。

需要强调的是，$\eta_{\mathrm{comp}}$ 依赖补偿前后风险变化，具有事后诊断性质。为避免循环论证，本文不将 $\eta_{\mathrm{comp}}$ 直接作为课程晋级门控的主条件，而主要用于结果分析中解释补偿动作是否有效。课程晋级中的补偿风险仍优先由不依赖事后风险下降的安全边界与动作质量指标构成，如高力矩占比、关节限位接近比例、动作 jerk 和相位对齐程度。

安全边界指标包括高力矩占比、关节限位接近比例和动作 jerk：

```math
P_{\tau}=\frac{1}{T}\sum_t
\mathbf{1}(|\tau_t|>\rho \tau_{\max})
```

```math
P_q=\frac{1}{T}\sum_t
\mathbf{1}(d(q_t,q_{\mathrm{limit}})<\epsilon)
```

```math
J_a=\frac{1}{T}\sum_t
\|a_t-2a_{t-1}+a_{t-2}\|^2
```

若踝关节动作幅值略有上升，但其主要集中在 touchdown 或 slip 后的有效恢复窗口内，同时伴随 slip ratio、roll/pitch RMS、action jerk 和 recovery time 降低，则可解释为必要的 compliant compensation，而非 aggressive compensation。

**什么叫 aggressive compensation？**

我建议把 aggressive compensation 定义为同时满足以下特征的补偿：

1. 发生在非有效恢复窗口；
2. 不能显著降低后续 contact risk 或 posture risk；
3. 伴随高 jerk、高 torque saturation 或接近 joint limit；
4. 在 terrain transition 后持续时间过长。

也就是说，你不要说“力矩大就是危险”，而要说：

> **无效、高频、非相位对齐且接近硬件边界的补偿才是不安全补偿。**

建议补偿风险改写成：

​												$$ R_{comp} = \alpha P_\tau + \beta P_q + \gamma J_a + \delta\left(1-C_{phase}\right) $$

其中 $C_{phase}$ 越高越好，因此在风险函数中使用 $1-C_{phase}$ 表示非相位对齐补偿风险。$\eta_{\mathrm{comp}}$ 越高越好，但它主要作为诊断指标用于解释补偿有效性，不直接放入 $R_{comp}$ 的门控公式。这样可以避免用“风险下降”反过来定义“安全补偿”的循环论证，使补偿风险更多依赖实时可统计的安全边界与动作质量指标。

### 4.4 接触—姿态—补偿风险链条诊断

这部分要服务于机制解释，而不是指标堆砌。你要证明：

```
risk gate 首先降低 contact risk；
contact risk 降低进一步降低 posture risk；
补偿动作可能不一定更小，但更相位对齐、更短时、更远离安全边界。
```

## 5 试验设计

### 5.1 对比方法

为验证所提方法不是简单“参数调得更细”或“训练得更久”，设置如下 5 组对比方法。

| 组别 | 方法 | 目的 |
|---|---|---|
| B0 | Flat / Rigid Policy Transfer | 证明草地类地形任务具有难度 |
| B1 | Coupled Randomization | 证明无课程耦合随机化训练不稳定 |
| B2 | Fixed-schedule Terrain Curriculum | 对照按固定训练步数升级的普通课程 |
| B3 | Success-rate-gated Curriculum | 关键 baseline，检验 risk gate 相比 success-rate gate 的贡献 |
| Ours | Risk-gated Curriculum | 本文方法 |

所有课程类方法共享相同地形阶段、最终草地参数分布、训练总步数、PPO 超参数、策略网络、奖励函数、速度指令范围、episode 长度、终止条件和 evaluation seeds。唯一差异在于课程晋级准则。

### 5.2 评价指标

评价指标分为总体性能和风险诊断两类。总体性能包括成功率、平均行走距离、速度跟踪误差和摔倒次数。风险诊断指标包括：touchdown error、foot slip ratio、unexpected contact、missed/delayed support、contact-window IoU、base roll/pitch RMS、base angular velocity、recovery time、torque peak、torque saturation ratio、joint limit margin、action jerk 和 compensation efficiency。

### 5.3 预期结果

相比 Success-rate-gated Curriculum，本文方法不一定在所有补偿幅值指标上取得最低值，但应在相近成功率下表现出更低的接触风险、更快的姿态恢复和更安全的补偿模式。尤其是在 flat-to-grass transition 和高扰动草地场景中，本文方法预期能够显著降低 touchdown timing error、foot slip ratio 和 missed/delayed support，并提高 contact-window IoU，从而切断 grass perturbation 到 contact mismatch 的前端失稳链条。

## 6 结果与讨论拟写逻辑

结果部分应优先围绕三个问题展开。第一，在相同地形难度和训练预算下，risk-gated progression 是否比 success-rate-gated progression 产生更低的 contact risk。第二，contact risk 的降低是否进一步减弱姿态扰动，如 roll/pitch RMS 和 recovery time。第三，若 torque peak 或 ankle action amplitude 未显著下降，甚至略有升高，补偿是否更相位对齐、更短时、更少高频振荡，并且更少接近 torque saturation 或 joint limit。

讨论部分需要强调，本文方法并不是简单抑制关节动作，而是使关节补偿重新分配到触地和打滑后的有效恢复窗口内。因而，补偿幅值本身不应作为唯一安全判据；更重要的是补偿动作是否降低后续接触风险和姿态风险，以及是否远离硬件安全边界。

## 7 结论

本文面向草地类柔顺不平整地形提出一种人形机器人风险门控课程强化学习方法。通过构建柔顺-不平整-低摩擦耦合地形随机化模型，并以地形参数难度驱动课程训练，方法能够在不依赖视觉和显式地形识别的条件下提升人形机器人本体感知鲁棒行走能力。引入接触、姿态和补偿三类风险作为课程晋级门控与诊断评价指标，可避免策略仅凭成功率进入高难度地形并形成剧烈补偿行为。后续工作将进一步开展实机草地测试，并研究更真实的草地沉陷模型与跨季节、跨湿度地面泛化能力。
