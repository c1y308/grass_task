# G1 草地类风险门控课程实验设计

## 目的

本文档定义一套面向《机械工程学报》中文论文的实验方案，研究内容为：人形机器人在草地类柔顺、不平整、低摩擦地形上的本体感知鲁棒行走。实验实现平台为 `unitree_rl_lab`，机器人任务为 Unitree G1 29 自由度行走任务。

## 研究问题

在不依赖视觉和显式地形识别的条件下，如何基于 Unitree G1 的本体感知反馈，通过草地类柔顺-不平整-低摩擦耦合地形随机化与风险门控课程学习，提升人形机器人在未知草地类地形上的鲁棒行走能力？

## 研究假设

H1：相比固定进度地形课程和成功率门控课程，所提出的风险门控课程能够降低草地类地形上的接触失配，具体表现为更低的触地时序误差、足底滑移比例、支撑相持续时间偏差、意外接触次数、漏支撑/延迟支撑比例，以及更高的期望接触窗口与真实接触窗口一致性。

H2：接触失配的降低会进一步改善姿态稳定性，具体表现为更低的机身 roll/pitch RMS、机身角速度、质心高度波动，以及更短的 flat-to-grass 地形过渡后恢复时间。

H3：本文方法不需要追求绝对踝关节动作幅值或力矩峰值最小。更重要的是形成更安全的补偿质量，包括更好的相位对齐、更低的动作 jerk、更低的力矩饱和比例、更大的关节限位裕度、更短的补偿持续时间，以及更好的事件后风险降低效果。事件后风险降低效果仅作为诊断指标使用，不作为课程晋级门控的主条件。

## 方法名称

完整名称：地形参数驱动的风险门控课程学习与风险诊断评价方法。

实验文件中的短名称建议使用：`g1_grass_risk_curriculum`。

## 备选设计方案

### 方案 A：纯风险驱动课程

风险指标直接更新地形分布。该方案表达能力强，但对当前论文目标而言过于复杂，且容易被审稿人批评为手工调参式训练。

### 方案 B：地形参数驱动课程 + 风险门控晋级

地形难度由外生地形参数向量和标量难度等级控制。风险指标只决定策略是否能够进入下一课程阶段，并用于诊断失败模式。本文选用该方案，因为它能够更清楚地隔离论文贡献。

### 方案 C：仅使用风险奖励塑形

将风险项直接并入 reward penalty。该方案实现简单，但论文贡献容易被解释为 reward tuning。可少量用于稳定训练，但不应成为本文主张的核心机制。

## 选定架构

实现时不重写上游 locomotion 代码，而是在 `unitree_rl_lab` 内新增一个研究扩展模块。该扩展模块包含：

- 草地类耦合地形参数配置；
- 地形参数驱动的课程阶段表；
- 风险指标计算；
- 风险门控晋级逻辑；
- baseline 任务变体；
- 训练与评估脚本；
- 结果聚合和论文表格导出。

策略观测保持本体感知-only。actor policy 不允许使用 height scanner 或视觉地形特征。若要使用特权地形参数，只能作为 critic-only 诊断信息，并且必须在论文中清楚区分；首轮推荐实验中，actor 和论文评价口径都保持 proprioception-only。

## 草地类地形模型

定义地形参数向量：

```math
\xi=\{h,\mu,k,c,p_{\mathrm{trans}}\}
```

参数含义：

- `h`：地形高度扰动，表示隐藏凸起、小坑和草面覆盖下的小尺度不平整；
- `mu`：摩擦系数，表示干草、湿草和泥草的摩擦差异；
- `k`：接触刚度或柔顺性，表示软到硬的草地接触变化；
- `c`：接触阻尼或沉陷耗散；
- `p_trans`：地形切换概率，表示 flat-to-grass 和 grass-to-flat 等过渡事件。

定义课程难度标量：

```math
\lambda \in [0,1]
```

难度随 `\lambda` 增加而提高：

```math
h_{\max}(\lambda)\uparrow,\quad
\mu_{\min}(\lambda)\downarrow,\quad
k_{\min}(\lambda)\downarrow,\quad
p_{\mathrm{trans}}(\lambda)\uparrow
```

## 课程变体

所有课程变体共享相同的地形阶段、最终地形参数分布、PPO 设置、网络结构、奖励项、总环境步数、评估随机种子、速度命令范围、episode 长度和终止条件。

唯一允许不同的是课程晋级准则。

### B0：Flat / Rigid Transfer

只在刚性平地训练，在草地类地形上评估。目的：证明任务本身具有难度。

### B1：Coupled Randomization

从训练一开始就在 `h`、`mu`、`k`、`c` 和 `p_trans` 全范围内随机化。目的：证明直接全范围耦合随机化不如课程学习稳定。

### B2：Fixed-Schedule Terrain Curriculum

只按固定训练步数推进地形阶段。目的：对比普通课程学习。

### B3：Success-Rate-Gated Curriculum

成功率超过阈值后推进地形阶段。目的：作为最关键 baseline，用来证明本文贡献不是单纯来自“有 gate”“训练更久”或“阶段更细”。

### Ours：Risk-Gated Curriculum

当地形阶段同时满足成功率和风险暴露阈值时才晋级：

```math
S_i \geq S^*,\quad
R_{\mathrm{contact}} \leq R_{\mathrm{contact}}^*,\quad
R_{\mathrm{posture}} \leq R_{\mathrm{posture}}^*,\quad
R_{\mathrm{comp}} \leq R_{\mathrm{comp}}^*
```

## 风险指标

### 接触风险

主要指标：

- touchdown timing error；
- foot slip ratio；
- stance duration deviation；
- unexpected contact count；
- missed/delayed support ratio；
- contact-window IoU。

定义锚点：

```math
E_{\mathrm{contact}}=
\left|t_{\mathrm{touchdown}}^{\mathrm{real}}-
t_{\mathrm{touchdown}}^{\mathrm{expected}}\right|
```

采用期望接触窗口定义每条腿在相位 $\phi$ 下的期望支撑时间段：

```math
W_l^{exp}=[t_{l,on}^{exp},t_{l,off}^{exp}]
```

真实接触窗口由法向接触力阈值得到：

```math
W_l^{real}=\{t\mid F_{l,t}^z>F_{min}\}
```

接触风险由四类失配组成：

```math
R_{contact}=w_1E_{td}+w_2E_{slip}+w_3E_{unexpected}+w_4E_{missed}
```

其中：

```math
E_{missed}=
\frac{1}{|\mathcal{W}^{exp}|}
\sum_{t\in\mathcal{W}^{exp}}
\mathbf{1}(F_{foot,t}^z<F_{min})
```

$E_{unexpected}$ 描述期望支撑窗口之外的提前接触或异常接触；$E_{missed}$ 描述期望支撑窗口内接触不足、延迟支撑或漏支撑。二者分别覆盖“过早接触”和“该支撑时未有效支撑”的接触失配。

作为补充诊断，报告 contact-window IoU：

```math
\mathrm{IoU}_{contact}=
\frac{|W_l^{exp}\cap W_l^{real}|}
{|W_l^{exp}\cup W_l^{real}|}
```

该指标越高表示真实接触越集中于期望支撑窗口内。IoU 主要用于评价分析，不强制作为课程晋级门控项。

### 姿态风险

主要指标：

- base roll RMS；
- base pitch RMS；
- base angular velocity peak/RMS；
- CoM height fluctuation；
- recovery time after terrain transition。

定义锚点：

```math
E_{\mathrm{posture}}=
\mathrm{RMS}(\theta_{\mathrm{roll}})
+\mathrm{RMS}(\theta_{\mathrm{pitch}})
+\max|\dot{\theta}_{\mathrm{base}}|
```

### 补偿风险

幅值类指标：

- mean ankle action amplitude；
- max ankle action amplitude；
- torque peak；
- torque RMS。

安全边界类指标：

- torque saturation ratio；
- joint limit margin；
- joint velocity peak；
- action rate；
- action jerk。

有效性类指标：

- compensation phase alignment；
- compensation efficiency；
- post-compensation slip reduction；
- post-compensation roll/pitch recovery；
- recovery time；
- high-frequency action energy。

定义：

```math
C_{\mathrm{phase}}=
\frac{\text{compensation energy within valid stance-response window}}
{\text{total compensation energy}}
```

```math
\eta_{\mathrm{comp}}=
\frac{\Delta R_{\mathrm{contact}}+\Delta R_{\mathrm{posture}}}
{E_{\mathrm{joint}}}
```

其中，$\eta_{\mathrm{comp}}$ 采用事件触发的固定时间窗计算。以 slip、unexpected contact 或 missed support 事件时刻 $t_0$ 为中心：

```math
\mathcal{T}_{pre}=[t_0-\Delta t_{pre},t_0),\quad
\mathcal{T}_{post}=[t_0,t_0+\Delta t_{post}]
```

```math
\Delta R_{\mathrm{contact}}=
\bar{R}_{\mathrm{contact}}(\mathcal{T}_{pre})
-
\bar{R}_{\mathrm{contact}}(\mathcal{T}_{post})
```

```math
\Delta R_{\mathrm{posture}}=
\bar{R}_{\mathrm{posture}}(\mathcal{T}_{pre})
-
\bar{R}_{\mathrm{posture}}(\mathcal{T}_{post})
```

$\eta_{\mathrm{comp}}$ 只用于解释补偿是否有效，不直接进入 $R_{comp}$ 的门控公式。课程晋级中的补偿风险使用不依赖事后风险下降的安全边界与动作质量指标：

```math
R_{comp}=\alpha P_\tau+\beta P_q+\gamma J_a+\delta(1-C_{phase})
```

```math
P_\tau=\frac{1}{T}\sum_t \mathbf{1}(|\tau_t|>\rho\tau_{\max})
```

```math
J_a=\frac{1}{T}\sum_t\|a_t-2a_{t-1}+a_{t-2}\|^2
```

## 训练矩阵

每种方法至少使用 3 个随机种子；若算力允许，使用 5 个随机种子。

| 方法 | 随机种子 | 训练步数 | 评估 episodes | 关键对比 |
|---|---:|---:|---:|---|
| B0 Flat/Rigid | 3 | 与 Ours 相同 | 每个 seed >= 200 | 任务难度 |
| B1 Full randomization | 3 | 与 Ours 相同 | 每个 seed >= 200 | 无课程 |
| B2 Fixed schedule | 3 | 与 Ours 相同 | 每个 seed >= 200 | 普通课程 |
| B3 Success-rate gate | 3 | 与 Ours 相同 | 每个 seed >= 200 | 最强 gate baseline |
| Ours Risk gate | 3 | 与 Ours 相同 | 每个 seed >= 200 | 主方法 |

## 评估场景

所有训练策略都使用完全相同的 held-out terrain seeds 评估。

场景组：

- `eval_flat_to_grass`：平地到草地过渡；
- `eval_mild_grass`：中等不平整、中等摩擦、中等柔顺；
- `eval_wet_grass`：低摩擦 + 中等不平整；
- `eval_soft_grass`：高柔顺性和阻尼变化；
- `eval_hard_hidden_bumps`：较高高度扰动 + 正常摩擦；
- `eval_extreme_coupled`：高高度扰动、低摩擦、低刚度、高地形切换概率。

## 真实 G1 机器人安全边界

真实 G1 实验是最终验证阶段，不是第一验证阶段。

安全规则：

- 首先完成 Isaac Lab 训练和评估；
- 然后使用 `unitree_rl_lab` 部署路径进行 Mujoco sim2sim；
- 首先使用低速命令；
- 尽可能使用保护架、吊带或人工扶持；
- 若出现重复打滑、过大 base roll/pitch、力矩饱和警告、关节限位警告或控制器通信不稳定，立即停止；
- 在户外草地前，先在安全人工草皮或软垫上进行短时受控测试。

真实机器人指标：

- 无安全停止完成试验；
- 视频标注的可见滑移次数；
- onboard log 中的 base roll/pitch RMS；
- 草地过渡后的恢复时间；
- 若日志可用，记录 action jerk 和 torque saturation ratio。

## 成功标准

实验包成功的标准：

- 五种方法变体都能从命名任务配置启动训练；
- 评估运行能产生 schema 完全一致的可比 CSV 文件；
- Ours 能与 B3 在完全相同地形阶段和训练预算下直接比较；
- 风险指标表能自动生成，并包含 missed/delayed support 与 contact-window IoU；
- 补偿风险门控不直接使用 $\eta_{\mathrm{comp}}$，$\eta_{\mathrm{comp}}$ 仅作为事件窗口诊断指标；
- 至少一张 event-aligned 图能说明补偿是否必要且是否相位对齐；
- 论文能在不手工数日志的情况下报告 contact risk、posture risk 和 compensation risk。
